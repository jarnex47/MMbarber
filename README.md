# MM Barber site

Landing page for MM Barber – built with Next.js and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

- `src/app/` – Layout, page, and global styles
- `src/components/` – Header, Hero, Services, About, CTAButton, Footer
- `public/` – Static assets (images, favicon)

## Fresha Booking

Update the Fresha booking URL in `src/components/CTAButton.tsx` with your actual booking link.
